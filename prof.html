Мои навыки

HTML (Hypertext Markup Language):

HTML - это основной язык разметки веб-страниц. Веб-разработчик должен знать синтаксис HTML для создания структуры и содержания веб-страницы.

CSS (Cascading Style Sheets):

CSS используется для стилизации веб-страниц и определения внешнего вида элементов. З

JavaScript:

JavaScript - это язык программирования, который используется для добавления интерактивности на веб-страницы. Навыки JavaScript позволяют создавать динамические элементы, обработку событий и многое другое.

React:

React - это библиотека JavaScript, разработанная и поддерживаемая Facebook, которая используется для создания пользовательских интерфейсов (UI) для веб-приложений. Вот некоторые ключевые аспекты React:

SCSS (Sassy CSS)::

SCSS - это синтаксический вариант Sass, который ближе к синтаксису обычного CSS.

Sass (Syntactically Awesome Style Sheets):

Sass - это препроцессор CSS, который предоставляет расширенный синтаксис для стилизации веб-страниц.
